# finetech_task

# Getting Started
1-Clone the repository:
git clone https://github.com/mostafamoham3d/finetech_task.git

2-Install dependencies:
flutter pub get
3-Run the app:
flutter run
# Requirements
Flutter 3.x.x or higher
Dart SDK
# Folder Structure
lib/: Application logic
assets/: Static files (images, fonts, etc.)
